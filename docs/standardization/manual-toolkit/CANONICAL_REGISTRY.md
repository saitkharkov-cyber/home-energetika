# Canonical Registry — документальный снимок

Этот файл содержит читаемый снимок утверждённых канонов.

Фактический источник production-состояния:

```text
oc_canonical_attributes
```

При расхождении между этим документом и production-базой сначала выполняется аудит. Автоматически считать документ источником истины нельзя.

## Поля

| Поле | Назначение |
|---|---|
| canonical_code | постоянный смысловой код |
| attribute_id | канонический OpenCart attribute_id |
| name | утверждённое название |
| group | текущая группа OpenCart |
| value_type | тип значения |
| format | канонический формат |
| allowed_values | допустимые значения, если это перечисление |
| status | draft / active |
| locked | защищён ли канон от случайной замены |
| verified_at | дата последней сверки |

## Правила ведения

1. Новая строка добавляется только после смыслового решения человека.
2. Изменение `attribute_id` требует отдельного решения.
3. Единицы измерения хранятся в названии атрибута.
4. Значения хранятся без единицы, если это предусмотрено каноном.
5. Для перечислений указываются допустимые значения.
6. Для числовых характеристик указываются формат и диапазон, если он утверждён.
7. `active + locked` означает утверждённый канон.
8. Этот файл не заменяет production-log.
