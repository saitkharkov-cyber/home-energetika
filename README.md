# Home-Energetika

## О проекте

**Home-Energetika** — репозиторий, объединяющий независимые подсистемы, предназначенные для развития интернет-магазина инженерного оборудования.

Подсистемы имеют собственную архитектуру, документацию и жизненный цикл, но используют общий каталог OpenCart.

---

# Подсистемы

## 1. Pump Selector

Интеллектуальный подборщик насосного оборудования.

Основные задачи:

* подбор насосов по рабочим параметрам;
* алгоритмы расчета;
* пользовательский интерфейс;
* интеграция с OpenCart.

**Точка входа в документацию:**

```text
pump-selector/PROJECT_MASTER_SUMMARY.md
```

---

## 2. Catalog Standardization

Подсистема управления качеством каталога.

Основные задачи:

* стандартизация характеристик товаров;
* унификация атрибутов;
* импорт данных производителей;
* генерация SQL;
* поддержание единого стандарта хранения данных.

**Точка входа в документацию:**

```text
catalog-standardization/PROJECT_MASTER_SUMMARY.md
```

---

# Архитектурный принцип

Каждая подсистема отвечает только за собственную предметную область.

Например:

* **Catalog Standardization** обеспечивает качество и единообразие данных каталога.
* **Pump Selector** использует уже стандартизированные данные и не занимается их исправлением.

Такое разделение позволяет независимо развивать обе подсистемы, сохраняя единый стандарт данных.

# Документация

Документация проекта построена иерархически.

1. `README.md`
2. `PROJECT_MASTER_SUMMARY.md`
3. `PROJECT_MASTER_SUMMARY.md` выбранной подсистемы
4. Остальные документы соответствующей подсистемы

---

# Основной принцип разработки

Развитие проекта строится по следующей последовательности:

**Архитектура → Стандарты → Данные → Функциональность**

Документация рассматривается как часть архитектуры проекта и развивается одновременно с кодом.

---

# Статус проекта

Проект находится в активной разработке.

Архитектура, стандарты и документация постепенно развиваются вместе с подсистемами. Новые компоненты и документы добавляются по мере развития проекта без нарушения принятых архитектурных принципов.
