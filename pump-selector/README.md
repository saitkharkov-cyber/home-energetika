# Pump Selector

## Назначение

**Pump Selector** — подсистема Home-Energetika, предназначенная для интеллектуального подбора насосного оборудования по заданным параметрам.

Подсистема использует стандартизированные данные каталога OpenCart и предоставляет пользователю рекомендации по выбору оборудования.

# Взаимодействие с другими подсистемами

Pump Selector использует результаты работы подсистемы **Catalog Standardization**.

Подборщик не занимается очисткой, исправлением или нормализацией данных каталога. Он работает только со стандартизированными характеристиками товаров.

---

# Основные задачи

Подсистема отвечает за:

- подбор оборудования по заданным параметрам;
- реализацию алгоритмов подбора;
- бизнес-логику рекомендаций;
- интеграцию с OpenCart;
- пользовательский интерфейс модуля.

---

# Архитектурный принцип

Pump Selector не зависит от конкретного производителя оборудования.

Алгоритмы работают с унифицированными характеристиками, предоставляемыми подсистемой Catalog Standardization.

Это позволяет расширять поддержку новых производителей и категорий оборудования без изменения базовой архитектуры подборщика.

---

# Статус проекта

Подсистема находится в рабочем состоянии и используется в составе Home-Energetika.

Дальнейшее развитие предполагает расширение поддерживаемых категорий оборудования и совершенствование алгоритмов подбора без нарушения существующей архитектуры.
