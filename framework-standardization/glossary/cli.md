# CLI

`CLI` — это command-line interface, то есть интерфейс командной строки.

В рамках `framework-standardization` CLI означает команду или утилиту, которую запускают из консоли, например через PowerShell.

CLI-команда может:

* читать данные;
* выводить отчёт в консоль;
* генерировать предложения;
* запускать проверки;
* создавать или обновлять файлы, если это явно разрешено задачей.

CLI сама по себе не означает, что разрешено менять данные, cache, production или выполнять SQL apply.

Для каждой CLI-задачи должны быть явно указаны:

* цель команды;
* входные параметры;
* формат вывода;
* разрешённые действия;
* запрещённые действия;
* readonly-ограничения, если команда работает с БД;
* проверки, которые нужно выполнить.

Если CLI-команда предназначена только для анализа, она не должна менять файлы, БД, cache или production-данные.
