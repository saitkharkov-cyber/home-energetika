# Catalog Standardization

## Назначение

**Catalog Standardization** — подсистема Home-Energetika, отвечающая за стандартизацию, контроль качества и импорт характеристик товаров в каталог OpenCart.

Подсистема обеспечивает единый стандарт хранения данных независимо от производителя и категории оборудования.

---

# С чего начать

Документация изучается в следующем порядке:

1. `PROJECT_MASTER_SUMMARY.md`
2. `CATALOG_STANDARD.md`
3. `HANDOFF.md`
4. Остальные документы по мере необходимости.

# Общий процесс работы

Каждый производитель проходит одинаковый цикл обработки:

```text
Источник данных
      ↓
Парсинг
      ↓
Нормализация
      ↓
Entity Matching
      ↓
Стандартизация
      ↓
Контроль качества
      ↓
Генерация SQL
      ↓
Импорт в OpenCart
```

Подробное описание процесса приведено в `DATA_PIPELINE.md`.

---

# Принципы

* Сначала утверждается стандарт, затем изменяются данные.
* Все массовые изменения должны быть воспроизводимыми.
* Каждое решение документируется.
* Временные файлы не должны попадать в репозиторий.
* Скрипты должны быть повторно используемыми и не зависеть от конкретного производителя, если это возможно.

---

# Основная цель

Создать универсальную систему стандартизации каталога, пригодную для обработки новых производителей и новых категорий оборудования с минимальными изменениями процесса.
